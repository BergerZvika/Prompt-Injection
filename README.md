# Prompt Injection Reference

An interactive single-page reference covering prompt injection attack techniques and defenses, built as a standalone HTML file.

## Contents

The page is organized into tabs covering:

| Category | Topics |
|----------|--------|
| **Direct Attacks** | Basic Injection, DAN Attack, Prompt Leaking, Structured Attack, Roleplay Bypass, Virtualization, Multi-Turn, Payload Splitting, Encoding/Obfuscation, Delimiter Confusion, Link Smuggling, Image Attack |
| **Indirect Attacks** | Indirect Injection (via external content) |
| **Defense** | Mitigations and countermeasures |

## Usage

Open `index.html` in any modern browser — no server or dependencies required.

## Purpose

Educational reference for understanding prompt injection vulnerabilities in LLM-based applications, intended for security researchers, red teamers, and developers building AI systems.
